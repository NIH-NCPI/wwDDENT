Welcome to the DDENT demo website. This website is intended to demonstrate how one might use FHIR to store associations between dataset variables and the relevant Concept Unique Identifiers (CUIs) identified via running an NLP extraction on the variable's description.

The website uses only FHIR to store data and currently relies solely on the following resource types:

  • CodeSystem
  • ValueSet
  • ConceptMap

To get started, feel free to navigate over to the DbGAP Study Page. Once there, you are welcome to explore any of the datasets have already been loaded including the ability to click on links to view translation functionality for various IDs that have been added to this system. This site is also open to permit users to load additional studies from DbGAP using accession IDs.

In addition to the ConceptMaps, users can also look around at the CodeSystems and ValueSets.
